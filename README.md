# FrameUI

FrameUI is a library for creating GUI layouts in Minecraft using item frames and maps. It allows you to create custom screens with a variety of components, including buttons, text, etc.

## Installation

To install FrameUI, simply download the latest release from the releases page and place it in your server's `plugins` directory. FrameUI requires Spigot 1.19.x to run.

## Usage

To use FrameUI, you will need to create a new `Screen` using the `Screen.Builder` class. The `Screen.Builder` allows you to specify the background image, size, and location of the screen, as well as the components to display on the screen.

## Customization
FrameUI provides a number of customization options to allow you to create the perfect GUI for your project. You can create custom components to display any content you like, or use the provided components to display buttons, text, etc.

You can also customize the appearance of the screen by setting the background image and adjusting the size and location of the screen.
