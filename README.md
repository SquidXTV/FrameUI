# FrameUI
The FrameUI **plugin** provides a way for plugin developers to easily create a **Screen** 
in the minecraft world.

A **Screen** consists of `width x height` **item frames** containing **maps** with
a custom renderer. 

This makes it possible to create **any kind of image rendering** in **vanilla** minecraft possible.
It also provides ways to handle **clicks** and **scrolls** on screens.
Creation of a **Screen** is done by either **XML** or manually via code.
